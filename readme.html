This app creates, modifies, and deletes information that it creates on the PAN Firewall. As a result, please do not edit addresses in PAN created by Phantom. Doing so may result in unexpected behavior when performing actions on Phantom-created addresses.

The app uses HTTP/ HTTPS protocol for communicating with the Tala server. Below are the default ports used by Splunk SOAR.
        Service Name Transport Protocol Port
        http tcp 80
        https tcp 443
